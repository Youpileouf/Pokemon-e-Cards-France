Ces cartes ne sont compatibles qu'avec le projet de restauration des fonctionnalités japonaises de Pokémon Emeraude disponible ici : https://github.com/Youpileouf/pokeemerald



Pour les utiliser, il suffit de débloquer la fonctionnalité CADEAU MYST.

Cette fonctionnalité se débloque dans le magasin de ROSYERE en complétant le formulaire :
CONNEXION AVEC LES DRESSEUR



Une fois CADEAU MYST. débloqué, lancer CADEAU MYST. puis connecter une seconde GBA avec un e-Reader via câble link.
